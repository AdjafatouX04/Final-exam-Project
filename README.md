# African Heroes Platform & Store
### [Live Deployment](https://final-exam-project.vercel.app) (Requires Setup)

## Project Overview
A comprehensive digital experience featuring:
1.  **Digital Museum of African Legends:** An immersive, high-tech tribute to African history.
2.  **Heritage & Heroes Store:** A fully functional e-commerce store with unique merchandise.
3.  **Admin Dashboard:** A secured panel to manage content.

## Deployment
This project is optimized for **Vercel**.
1.  Push to GitHub.
2.  Import project in Vercel.
3.  Deploy (Zero Config required).
